ФАЙЛЫ ДАННЫХ ДЛЯ СИСТЕМЫ ФАРМАМОНИТОР
===============================

В этой директории находятся примеры CSV-файлов с данными о пациентах для анализа нежелательных реакций на лекарственные препараты:

1. patient_data_example.csv - полный набор данных, включающий:
   - Идентификатор пациента
   - Возраст
   - Пол (M/F)
   - Название лекарственного препарата
   - Дозировка
   - Длительность приема (дни)
   - Нежелательная реакция
   - Тяжесть реакции (Легкая/Средняя/Тяжелая)
   - Сопутствующие заболевания
   - Дополнительные препараты

2. patient_data_simplified.csv - упрощенный набор данных, включающий:
   - Идентификатор пациента
   - Название лекарственного препарата
   - Нежелательная реакция
   - Возраст
   - Пол (M/F)

3. patient_data_minimal.csv - минимальный набор данных, включающий:
   - Идентификатор пациента
   - Название лекарственного препарата
   - Нежелательная реакция

КАК ИСПОЛЬЗОВАТЬ
---------------

1. Перейдите во вкладку "Анализ данных" в системе ФармаМонитор
2. Нажмите на область загрузки или кнопку "Выбрать файл"
3. Выберите один из CSV-файлов
4. Нажмите кнопку "Анализировать"

Система проанализирует файл и покажет сводную информацию о нежелательных реакциях, включая:
- Количество пациентов
- Число выявленных реакций
- Количество критических реакций
- Детальную информацию о каждой выявленной реакции
- Рекомендации по минимизации рисков

ПРИМЕЧАНИЕ
---------

Эти файлы содержат демонстрационные данные и предназначены только для тестирования функциональности системы.
